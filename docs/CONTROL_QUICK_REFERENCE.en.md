# Control Panel Quick Reference (EN)

## Before You Start
1. Open `control.html`.
2. Open `monitor.html` via **Open monitor**.
3. Confirm synchronization.
4. Adjust volume.

## Key Buttons
- **Pause/Start**: pause or resume simulation.
- **Reset**: return to baseline.
- **Start event**: trigger a clinical event.
- **Stop event**: stop the active event.
- **NIBP now**: immediate blood pressure measurement.
- **Start asystole / Stop asystole**: toggle asystole.

## Critical Toggles
- **Alarms**: enables alarm logic.
- **Sound**: enables audio.
- **ECG leads off / SpO2 off / TEMP off**: technical alarms.

## Frequent Adjustments
- **Patient type**: adult / pediatric / neonatal.
- **ECG gain** and **ECG sweep speed**.
- **Alarm volume**.

## Express Scenario (2 minutes)
1. Normal baseline (20 s).
2. Desaturation event (40 s).
3. SpO2 off (20 s).
4. Recovery (40 s).

## If Something Goes Wrong
1. Reopen the monitor with **Open monitor**.
2. Check **Sound** and **Alarm volume**.
3. Click or press a key to unblock browser audio.
