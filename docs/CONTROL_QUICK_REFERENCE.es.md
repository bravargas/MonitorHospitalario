# Referencia rápida del control (ES)

## Antes de empezar
1. Abrir `control.html`.
2. Abrir `monitor.html` con **Open monitor**.
3. Confirmar sincronización.
4. Ajustar volumen.

## Botones clave
- **Pause/Start**: pausa/reanuda simulación.
- **Reset**: vuelve a baseline.
- **Start event**: inicia evento clínico.
- **Stop event**: detiene evento.
- **NIBP now**: medición de presión inmediata.
- **Start asystole / Stop asystole**: alterna asistolia.

## Toggles críticos
- **Alarms**: activa lógica de alarmas.
- **Sound**: activa audio.
- **ECG leads off / SpO2 off / TEMP off**: alarmas técnicas.

## Ajustes frecuentes
- **Patient type**: adulto/pediátrico/neonato.
- **ECG gain** y **ECG sweep speed**.
- **Alarm volume**.

## Escenario express (2 minutos)
1. Baseline normal (20 s).
2. Desaturation (40 s).
3. SpO2 off (20 s).
4. Recuperación (40 s).

## Si algo falla
1. Reabrir monitor con **Open monitor**.
2. Verificar **Sound** y **Alarm volume**.
3. Hacer interacción (click/tecla) para desbloquear audio.
